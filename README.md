# BMI Calculator

Приложение для расчёта индекса массы тела (BMI) с визуальным результатом и цветовой индикацией.

## Функционал

- Выбор роста и веса (слайдеры)
- Расчёт BMI по формуле: вес / (рост × рост)
- Отображение результата с советом
- Цветовая индикация:
  - 🔵 Недостаток веса (<18.5) — синий
  - 🟢 Нормальный вес (18.5–24.9) — зелёный
  - 🔴 Избыточный вес (>24.9) — розовый
- Переход на экран результата
- Кнопка перерасчёта

## Технологии

- Swift
- UIKit
- Auto Layout
- MVC (Model-View-Controller)
- UISlider
- UIColor
- Segue

## Структура MVC

- **Model**: BMI.swift, CalculatorBrain.swift
- **View**: Main.storyboard
- **Controller**: CalculateViewController.swift, ResultViewController.swift

# BMI Calculator

Body Mass Index calculator app with visual result and color indication.

## Features

- Height and weight selection (sliders)
- BMI calculation: weight / (height × height)
- Result display with advice
- Color indication:
  - 🔵 Underweight (<18.5) — blue
  - 🟢 Normal weight (18.5–24.9) — green
  - 🔴 Overweight (>24.9) — pink
- Result screen transition
- Recalculate button

## Technologies

- Swift
- UIKit
- Auto Layout
- MVC (Model-View-Controller)
- UISlider
- UIColor
- Segue

## MVC Structure

- **Model**: BMI.swift, CalculatorBrain.swift
- **View**: Main.storyboard
- **Controller**: CalculateViewController.swift, ResultViewController.swift
